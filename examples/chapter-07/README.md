# 第 7 章示例：Tools 与 Agent

这一章的示例用一个最小版 `AI 学习助手` 演示 Tool 和 Agent 的关系。

## 你会学到什么

- 什么是 Tool，为什么 Tool 需要明确输入和输出
- Agent 如何根据用户问题决定调用哪个 Tool
- 为什么不是所有任务都适合 Agent

## 运行前提

- Python 3.10 或更高版本
- 不需要额外的第三方依赖

## 安装方式

这个示例不依赖外部包，直接运行即可。

## 运行命令

```bash
python3 main.py
```

## 预期输出

脚本会依次演示：

1. 查询课程大纲
2. 获取某一章的摘要
3. 生成练习题
4. 让 Agent 根据用户输入自动选择工具

## 学习重点

这个示例的重点不是模拟真实模型，而是把 Agent 的决策过程讲清楚。

你应该重点观察：

- 每个 Tool 只负责一件事
- Agent 会先判断问题类型，再决定调用哪个 Tool
- 当问题无法归类时，Agent 会给出澄清建议
