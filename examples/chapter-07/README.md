# 第 7 章示例：Tools 与 Agent

这一章的示例用一个最小版 `AI 学习助手` 演示 Tool 和 Agent 的关系，并补一版更接近真实 LangChain tool/message 接口的对照实现。

## 示例层级

- 层级：`教学版`
- 重点：看清 Tool 输入输出边界，以及 Agent 如何做最小决策。
- 边界：示例优先讲概念，不接真实模型规划能力，也不覆盖工具权限、安全隔离和多轮失败处理。
- 对照：脚本里额外提供 `真实框架版`，使用 `@tool`、`AIMessage.tool_calls` 和 `ToolMessage` 展示接近真实 agent loop 的接口形状。

## 你会学到什么

- 什么是 Tool，为什么 Tool 需要明确输入和输出
- Agent 如何根据用户问题决定调用哪个 Tool
- 为什么不是所有任务都适合 Agent

## 运行前提

- Python 3.10 或更高版本
- 不需要额外的第三方依赖

## 安装方式

这个示例不依赖外部包，直接运行即可。

## 运行命令

```bash
python3 main.py
```

## 预期输出

脚本会依次演示：

1. 查询课程大纲
2. 获取某一章的摘要
3. 生成练习题
4. 让 Agent 根据用户输入自动选择工具
5. 用真实框架消息结构执行同一批工具调用

## 学习重点

这个示例先把 Agent 的决策过程讲清楚，再让你看到真实框架里工具调用消息是怎样组织和回传的。

你应该重点观察：

- 每个 Tool 只负责一件事
- Agent 会先判断问题类型，再决定调用哪个 Tool
- 当问题无法归类时，Agent 会给出澄清建议
