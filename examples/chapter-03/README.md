# 第 3 章示例：第一个链式应用

这个示例对应课程第 3 章，目标是把 `PromptTemplate`、一个最小的“模型”函数和输出解析串成一条完整链路，做出 `AI 学习助手 V1`。

## 你将学到什么

- 如何把一次 AI 调用拆成输入、编排、输出三层
- 为什么链式结构比直接写一个大函数更容易维护
- 如何先做出一个可跑通的最小版本，再为后续 RAG 和 Agent 预留接口

## 运行前提

- Python 3.10+
- 只需要 `langchain-core`
- 不需要 API Key，也不需要额外服务

## 安装

在当前目录执行：

```bash
pip install -r requirements.txt
```

## 运行

```bash
python3 main.py
```

## 预期输出

程序会生成一份关于 `LangChain 核心抽象` 的学习计划，包含：

- 学习目标
- 核心概念
- 学习顺序
- 练习建议

## 你在学什么

这不是在学“一个聊天脚本”，而是在学一条可维护的 AI 流程：

`输入 -> PromptTemplate -> Runnable 模型 -> 输出解析 -> 结果`

示例代码使用了一个本地的模拟模型函数，重点是帮你先理解链式结构。后续你可以把这个函数替换成真实 LLM 调用。
