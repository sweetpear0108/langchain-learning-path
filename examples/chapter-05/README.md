# 第 5 章示例：RAG 入门

这一章的示例演示 `AI 学习助手` 的基础 RAG 骨架。

## 示例层级

- 层级：`教学版`
- 重点：理解 RAG 的最小流程，包括切分、检索、拼装上下文和回答。
- 边界：示例主要帮助你看清流程，不代表真实向量数据库、Embedding 服务或在线知识库接入方案。

你会学到：

- 怎么把课程资料切成小块
- 怎么根据问题找相关片段
- 怎么把检索结果组织成回答
- 为什么 RAG 比直接把整篇文档塞给模型更适合学习站

## 运行前提

- Python 3.10+
- 只需要本地运行，不需要 API key

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python3 main.py
```

## 预期输出

脚本会打印：

- 课程资料切分后的片段数量
- 每个问题检索到的相关片段
- 基于片段生成的简洁回答

## 你在学什么

这个示例不追求真实大模型调用，而是把 RAG 的关键流程拆开：

1. 准备知识库
2. 切分文档
3. 检索相关片段
4. 拼装上下文
5. 生成回答

这是 `AI 学习助手` 从“会聊天”升级为“会基于课程资料回答问题”的第一步。
