# TODO - Flat UI Redesign

本轮任务将文档站重构为更直接、干净、易读的扁平化界面语言。

执行原则：

- 先统一视觉基线，再扩展到具体页面
- 用排版、边框、色块和留白表达层级
- 减少阴影、渐变、玻璃感和装饰性动效
- 每次只完成一个可验证的改造单元

建议按顺序执行：`FD-01 -> FD-02 -> FD-03`

---

## FD-01 - 重构全站 Flat 视觉基线与首页信息架构

- [ ] 重写首页 hero、导航入口和学习路径区块
- [ ] 收敛全站主色、背景色、边框和文字层级变量
- [ ] 去除当前偏卡片化和轻拟物的阴影、模糊和渐变表达
- [ ] 将按钮、导航、链接反馈改为以色值变化为主的扁平交互
- [ ] 确保首页在桌面端和移动端都保持清晰分区

**完成标准**

- 首页第一屏更像信息海报而不是营销卡片集合
- 用户能快速判断学习入口、章节主线和项目边界
- 主题样式与 Flat Design 原则一致

**涉及文件**

- `docs/index.md`
- `docs/.vitepress/theme/custom.css`
- `docs/.vitepress/config.mts`

---

## FD-02 - 统一文档页的扁平化组件语言

- [ ] 统一正文标题、段落、列表、引用和表格的层级节奏
- [ ] 调整侧边栏、页内目录和翻页区块的平面化表现
- [ ] 让卡片、提示块、标签和链接列表使用统一描边与留白规则
- [ ] 收敛 hover、active、focus 的颜色反馈逻辑

**完成标准**

- 非首页页面也具备一致的 Flat 风格
- 信息层级主要由排版和间距表达，而不是阴影

**涉及文件**

- `docs/.vitepress/theme/custom.css`
- `docs/guide/*.md`
- `docs/faq/*.md`
- `docs/compare/*.md`
